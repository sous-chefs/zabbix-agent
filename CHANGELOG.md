# CHANGELOG

## Unreleased

- Enable circleci testing
- Change testing to dokken
- Require Chef Infra Client 14 or later to eliminate the need for the build-essential cookbook
- Replace usage of apt cookbook with the apt_update resource built into Chef Infra Client
- Replace Ubuntu 14.04 ChefSpecs with 16.04 to avoid deprecation warnings
- Remove unnecessary long_description metadata from the metadata.rb file
- Remove the ChefSpec coverage report as these reports are deprecated
- Simplify platform checks using platform? and platform_family? helpers
- Remove ChefSpec matchers as these are auto generated by ChefSpec now
- Removed the apt-update notification from the apt_repository resource as Chef Infra Client automatically performs the apt-update when the repo is updated
- Use multi-package installs for dependency installation to speed up Chef Infra Client runs.
- Don't include the yum::default recipe or require the yum cookbook on RHEL based systems
- Use platform_family in many locations to better support RHEL-like and Debian-like systems
